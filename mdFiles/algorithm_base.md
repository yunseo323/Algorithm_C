## 배열의 기본 개념(동적할당 이용하기)

배열을 선언할 때 요소 개수를 상수만 사용 가능하다(변수x)

필요할 때 메모리를 확보하고, 불필요해지면 메모리를 해제하면서 필요한 요소 개수의 배열을 만들 수 있음(동적할당)

calloc / malloc 함수

## *공백 포인터와 NULL

공백 포인터(null pointer)를 나타내는 것이 공백포인터 상수라고 하는 매크로 NULL이다.

NULL의 정의는 '값 0을 갖는 모든 정수, 상수 또는 상수식을 void *로 형 변환한 식' 이다.

## *난수 생성 단계

1. <time.h>선언
2. 난수의 seed를 초기화하기 위해 srand 함수를 호출한다. → srand(time(NULL))
3. 난수를 생성하기 위해 rand 함수를 호출함

*rand 함수가 생성하는 난수는 의사 난수임(난수처럼 보이지만 일정한 규칙에 따라 생성되는 수)

## c언어의 메모리 구조

1.데이터 영역

전역 / 정적(static) 변수가 할당되는 영역

프로그램을 시작하면 할당하고, 프로그램을 종료하면 메모리에서 해제함

2.스택 영역

함수 호출 시 생성되는 지역변수와 매개변수가 저장되는 영역

함수 호출이 완료되면 사라짐

3.힙 영역

필요에 따라 동적으로 메모리를 할당

힙 영역은 할당해야 할 메모리 영역의 크기를 프로그램이 실행되는 동안(run time)결정해야 하는 경우에 사용함. 관리가 가능한 데이터 외에 다른 형태의 데이터를 관리하기 위한 빈공간이다. 즉 동적할당을 통해 생성된 동적 변수를 관리하기 위한 영역.

데이터 영역과 스택 영역은 컴파일러가 미리 공간을 예측하고 할당할 수 있지만, 동적 변수는 어느 시점에 얼마만큼의 공간을 할당할지 정확하게 예측할 수 없으므로 프로그램 실행 중(runtime)에 결정한다.
